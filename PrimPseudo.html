Add the starting vertex to the priority queue: - Insert `(0, startNode, None)` into `minHeap` (weight, vertex, parent). 3. While the priority queue is not empty: - Extract the edge with the smallest weight from `minHeap`. - If the extracted vertex is not in `visited`: - Add the vertex to `visited`. - If the vertex has a parent (i.e., it's not the starting vertex), add the edge to `MST`. - For each adjacent vertex of the extracted vertex: - If the adjacent vertex is not in `visited`, add the edge to `minHeap`. 4.
